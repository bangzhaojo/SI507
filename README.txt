##RUNNING MY CODE##

Run my_app.py in CMD, then go to http://127.0.0.1:5000/ on a browser for the website.

The website is about Covid-19 data search. It contains all the covid-related data of over 3200 counties in the United States. You can filter and search any covid data as you want.
The first page of the website is a form, and it contains several filterer. Once you choose the your requirement of the covid data, you will get a table of all the data that you want. It contains death rate, infection rate, vaccination rate, poverty rate, minority rate data of all counties.


##DATA SOURCE##

read Data_Source_Description.pdf for details.

##DATA STRUCTURE##

All the data is processed in a tree. County nodes are children of State nodes.
